# Jagoskill

Jagoskill is a Laravel-based LMS application developed locally and deployed to production hosting from this repository.

## Local development

- App URL: `http://localhost:8088`
- Main working directory: `D:\rocket`

## Deployment direction

- Local development happens on laptop
- Source of truth is this Git repository
- Production target is `jagoskill.com`

## Notes

- Do not commit `.env`, database dumps, logs, or local credentials
- Use `.env.example` as the production setup template
- Keep production changes flowing from local development, not by editing files directly on hosting
